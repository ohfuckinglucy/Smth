Уплотнение позволяет избежать увеличения количества узлов с одинаковой степенью, 
тем самым сохраняя эффективность операций.

Алгоритм удаления узла с наименьшим приоритетом

    Удаление минимального узла:
        Находим и удаляем минимальный узел из кучи (узел, который будет возвращен как результат операции).
        Если у этого узла есть дочерние узлы, переместите их в список корней кучи.

    Обновление списка корней:
        Обновите указатель на минимальный узел кучи, если он не пуст.
        Убедитесь, что дочерние узлы минимального узла теперь находятся в списке корней и обновите их родительские указатели.
    
    Уплотнение списка корней:
        После добавления дочерних узлов в список корней необходимо провести уплотнение списка.
        Уплотнение — это процесс, в котором вы объединяете узлы с одинаковой степенью (числом детей) для уменьшения общего количества узлов в списке корней.
    
    
Уплотнение списка корней (consolidation)

    Инициализация массивов:
        Создайте массив указателей (или вектор) фиксированного размера, чтобы отслеживать узлы по их степени. Размер массива равен максимальной возможной степени узла в куче.
        
    Обход списка корней:
        Переберите все узлы в списке корней, для каждого узла определите его степень.
        Если узел с такой же степенью уже существует в массиве, объедините их, что означает, что один узел становится ребенком другого. Обновите массив с учетом нового узла.
        
    Создание нового списка корней:
        После завершения обхода создайте новый список корней из массива, исключая пустые элементы. 
        Укажите новый минимальный узел.
